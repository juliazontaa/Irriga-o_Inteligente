# Irriga-o_Inteligente
Desenvolvemos um sistema utilizando Arduino capaz de monitorar a umidade do solo e tomar decisões automaticamente.  Quando a terra está seca, o sistema ativa uma pequena bomba d’água, garantindo que a planta receba exatamente o que precisa.
